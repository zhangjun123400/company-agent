# 知识库

此目录存放内部历史技术文档和历史 PRD 文档，供智能体在生成技术可行性报告时参考。

## 使用方式

1. 将历史技术文档（.md / .txt）放入此目录
2. 将历史 PRD 文档（.md / .txt）放入此目录
3. 智能体会自动加载所有文件内容作为分析参考

## 文档类型建议

| 文件类型 | 示例 | 用途 |
|----------|------|------|
| 技术方案模板 | `tech-plan-template.md` | 统一技术方案输出格式 |
| 历史技术方案 | `2025-Q1-xxx方案.md` | 参考已有方案选型和架构 |
| 技术规范 | `coding-standards.md` | 确保方案符合内部规范 |
| 架构文档 | `system-architecture.md` | 了解现有系统约束 |
| 历史 PRD | `2025-xxx-PRD.md` | 相似需求的技术方案参考 |

## 格式要求

- 文件编码：UTF-8
- 文件格式：Markdown (.md) 或纯文本 (.txt)
- 文件大小：建议每个文件不超过 100KB

## 加载逻辑

智能体会在生成技术可行性报告时：
1. 扫描此目录下所有 .md 和 .txt 文件
2. 截取前 20000 字符（避免上下文溢出）
3. 作为 Claude 分析的参考上下文传入
